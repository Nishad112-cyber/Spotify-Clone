
# Spotify Clone

A Spotify-inspired web page built using **HTML, CSS, and Bootstrap**, focusing on styling, layout, and responsive design.

## Features

- Fully styled web page using CSS and Bootstrap
- Responsive design for desktop and mobile
- Includes navigation bar, playlists, and music cards
- Focus on CSS effects like hover, gradients, and layout

## Technologies Used

- HTML
- CSS
- Bootstrap 5
